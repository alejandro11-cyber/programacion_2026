1 DEFINE CLASES:
Una clase es una plantilla o modelo en programación orientada a objetos que define las características y comportamientos de un conjunto de objetos. Contiene atributos (datos o propiedades) y métodos (funciones que describen acciones)

2 DEFINE ENCAPSULAMIENTO:
El encapsulamiento es el principio de ocultar los detalles internos de una clase y controlar el acceso a sus atributos y métodos. Se logra usando modificadores de acceso como ,  o , y proporcionando métodos getters y setters para manipular los datos de forma segura.

3 DEFINE CONSTRUCTOR:
Un constructor es un método especial dentro de una clase que se ejecuta automáticamente al crear un objeto. Sirve para inicializar los atributos con valores específicos
